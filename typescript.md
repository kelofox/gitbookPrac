## WebStorm

### Hot Key：

指令					  |功能
----------------------|:--------------------------:
command + shift + f   |全專案搜索
command + b 			  |至有呼叫該function的行數
双击shift           	  |弹出小浮窗搜索所有
command+shift+U 		  |大小写转换
command+shift+加号/减号|收缩整个文档的代码块
shift+F6              |重命名

### 設定：

#### 橫向拖曳時會自動捲動：
Settings `>` Editor `>` General, Virtual Space `>` Show virtual space at file bottom
